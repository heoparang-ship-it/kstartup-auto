# 신혼생활 매칭 기준 (Single Source of Truth)

> 이 문서는 classify.py가 공고 티어를 판정하는 **유일한 기준 문서**입니다.
> 기준을 바꾸려면 이 파일을 수정하고, classify.py의 키워드를 동기화하세요.

## 정체성 (매칭 컨텍스트)

- **도메인**: sinhon.life
- **카테고리**: B2C 신혼부부 라이프스타일 플랫폼
- **기술**: Next.js + Claude API(AI 비서), PWA
- **BM**: 무료 + CPC 광고 + 제휴 수수료
- **소재지**: **인천** (허파랑 대표, XCom)
- **단계 (이중 트랙)**:
  - 트랙 A — 초기창업자 (허파랑 대표, XCom 업력 3년 이내)
  - 트랙 B — 예비창업자 (배우자 명의, 사업자등록 전 상태)

> 유튜브/크리에이터 활동은 이 매쩭의 컨텍스트가 아니다.

## 지역 매쩭 규칙 (🔴 판정의 핵심)

1. "전국" 폨함 → **통과**
2. "인천" 포함 (인천 단독 / 서울+경기+인천 / 수도권 통합 등) → **통과**
3. "비수도권" 포함 → **🔴** (인천은 수도권)
4. 서울 단독 / 경기 단독 / 기타 지역 단독 → **🔴**

서울 소재 시설 입주 공고는 본사 이전을 전제하므로 **🔴**. 단 프로그램만 참여하는 형태는 🟡.
