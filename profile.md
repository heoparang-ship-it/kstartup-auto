# 신혼생활 매칭 기준 (Single Source of Truth)

> 이 문서는 sinhon-kstartup-matcher 스킬이 공고 티어를 판정하는 **유일한 기준 문서**입니다.
> 기준을 바꾸려면 이 파일만 수정하세요. 다음 실행 시 전체 풀이 자동 재분류됩니다.

## 정체성 (매칭 컨텍스트)

- **도메인**: sinhon.life
- **카테고리**: B2C 신혼부부 라이프스타일 플랫폼
- **기술**: Next.js + Claude API(AI 비서), PWA
- **BM**: 무료 + CPC 광고 + 제휴 수수료
- **소재지**: **인천** (허파랑 대표, XCom)
- **단계 (이중 트랙)**:
  - 트랙 A — 초기창업자 (허파랑 대표, XCom 업력 3년 이내)
  - 트랙 B — 예비창업자 (배우자 명의, 사업자등록 전 상태)

> ⚠️ **유튜브/크리에이터 활동은 이 매칭의 컨텍스트가 아니다.** 대표자의 별도 채널 운영 경험이 있더라도, 공고 매칭 정체성은 "B2C 플랫폼 스타트업"으로만 판단한다.

## 지역 매칭 규칙 (🔴 판정의 핵심)

공고의 대상 지역 문자열을 다음 순서로 체크:

1. "전국" 포함 → **통과**
2. "인천" 포함 (인천 단독 / 서울+경기+인천 / 수도권 통합 등) → **통과**
3. "비수도권" 포함 → **🔴** (인천은 수도권)
4. 서울 단독 / 경기 단독 / 기타 지역(부산/대구/대전/광주/울산/강원/충청/호남/제주/경북/경남) 단독 → **🔴**

서울 소재 시설 입주 공고(예: 서울창업허브 공덕 입주)는 본사 이전을 전제하므로 **🔴** 처리. 단 "서울창업허브 오픈이노베이션"처럼 공간 이전 없이 프로그램만 참여하는 형태는 공고 본문 확인 후 🟡/🟢.

## 단계 매칭 규칙

- 공고가 "예비창업자" 대상 → 트랙 B(배우자)로 지원 가능 → **통과**
- 공고가 "초기창업자(업력 1~3년)" 대상 → 트랙 A(허파랑)로 지원 가능 → **통과**
- "후속지원", "업력 3년 초과", "Series A+" 등 → **🔴**

## 🟢 강력 매칭 (1순위)

- 범용 사업화, 모두의 창업, 유니콘 브릿지, 오픈이노베이션
- **예비창업패키지** (트랙 B 전용 — 배우자 명의 지원)
- **초기창업패키지** (트랙 A)
- 인천 지역 전용 공고 (인천창조경제혁신센터, 인천테크노파크, 남동구 청년창업지원센터 등) + 업종 제한 없음
- 예비/초기창업자 대상 + 업종 제한 없음 + 지역(전국 또는 인천 포함)

## 🟡 검토 매칭 (2순위)

- 액셀러레이팅, 청년창업, IP디딤돌, 사회적기업 (인천/전국)
- 콘텐츠·IT·AI 일반 카테고리 (인천/전국)
- 비대면 교육/설명회 (지역 무관)
- 전국 대상이나 일부 참여 요건이 애매한 경우

## 🟠 AI 포지셔닝 필요 (3순위)

- AX 버티컬, LLM, AI대전환, AI 바우처
- 판정 근거: Claude API 기반 AI 비서 기능을 보유 → "AX 수직화" 또는 "AI 버티컬 SaaS"로 리포지셔닝 가능
- 이 티어에 들어가면 대시보드 메모에 "AI 포지셔닝 필요" 명시

## 🔴 즉시 제외 (풀에 저장하지 않음)

**업종 한정**: 반도체/팹리스/바이오/농업/에너지/방산/로봇/항공우주/관광/게임/가구/세라믹/해운물류/패션/의료기기

**지역 한정**: 위 §지역 매칭 규칙 3~4항 해당 (인천 미포함 지역 전용 및 비수도권 전용)

**자격 한정**: 이민자(OASIS) / 여성전용 / 제대군인 / 자립청년 / 소상공인 전용

**단계 불일치**: 후속지원, IR/IPO 직전(Series B+ 전제), 해외진출 전제(수출실적 요건), 업력 3년 초과 대상

**성격 불일치**: 수행기관(운영기관) 모집, B2G 공공시장 진출 전용, 서울 소재 시설 입주 전용(본사 이전 요구)

## 티어 판정 우선순위

충돌 시 다음 순서로 판정한다:

1. 🔴 즉시 제외 조건 중 하나라도 맞으면 → red (다른 조건 무시)
2. 지역·단계·업종이 모두 부합하면서 🟢 키워드 매칭 → green
3. 지역·단계·업종은 맞지만 매칭 강도가 중간 → yellow
4. AI 관련 공고이고 2에서 탈락한 경우(업종 한정이 AI면 2 통과) → orange
5. 위 어느 것도 아니면 → red
