# 신혼생활 매칭 기준 (Single Source of Truth)

> 이 문서는 classify.py가 공고 티어를 판정하는 **유일한 기준 문서**입니다.
> 기준을 바꾸려면 이 파일을 수정하고, classify.py의 키워드를 동기화하세요.
>
> **버전**: v7.2 (2026-04-19) — false-red 교정 3종 반영

## 정체성 (매칭 컨텍스트)

- **도메인**: sinhon.life
- **카테고리**: B2C 신혼부부 라이프스타일 플랫폼
- **기술**: Next.js + Claude API(AI 비서), PWA
- **BM**: 무료 + CPC 광고 + 제휴 수수료
- **소재지**: **인천** (허파랑 대표, XCom)
- **단계 (이중 트랙)**:
  - 트랙 A — 초기창업자 (허파랑 대표, XCom 업력 3년 이내)
  - 트랙 B — 예비창업자 (배우자 명의, 사업자등록 전 상태)

> 유튜브/크리에이터 활동은 이 매칭의 컨텍스트가 아니다.

## 지역 매칭 규칙 (🔴 판정의 핵심)

1. "전국" 포함 → **통과**
2. "인천" 포함 (인천 단독 / 서울+경기+인천 / 수도권 통합 등) → **통과**
3. "비수도권" 포함 → **🔴** (인천은 수도권)
4. 서울 단독 / 경기 단독 / 기타 지역 단독 → **🔴**

서울 소재 시설 입주 공고는 본사 이전을 전제하므로 **🔴**. 단 프로그램만 참여하는 형태는 🟡.

## 🟢 강력 매칭 (1순위)

- 예비창업패키지, 초기창업패키지, 모두의 창업, 유니콘 브릿지, 사업화, 범용
- 인천 지역 전용 공고 (인천창조경제혁신센터, 인천테크노파크, 남동구 등)

### 🟢 인천 앵커 기관 우회 규칙 (false-red 방지, v7.2 신규)

다음 기관명이 공고 **제목**에 포함되면, 업종/자격 한정이 일부 걸리더라도 일단 🟢으로 올리고 공고 본문 검토 권장 플래그(risk_flag=industry_bypass)를 붙인다:

- **인천창조경제혁신센터 / ICCE / 인천TP / 인천테크노파크 / 인천스타트업파크**
- 이들은 단독 운영이든 파트너(예: ICCE × INNOPOLIS, 인천창조경제혁신센터 × 대한항공)든 sinhon.life 입장에서는 우선 검토 대상
- 단, 실제 모집 분야가 sinhon.life와 명확히 배치되면(예: 항공우주 엔진 개발) 사용자 판단으로 스킵 — 검토 기회를 잃지 않는 게 핵심

## 🟡 검토 매칭 (2순위)

- 액셀러레이팅, 청년창업, IP디딤돌, 사회적기업, 콘텐츠, 크리에이터, 1인미디어
- IT, 디지털, 설명회, 교육, 세미나, 네트워킹, 오픈이노베이션
- 경진대회, 해커톤, 데모데이, 피칭, IR, 팁스, TIPS, 글로벌, 투자연계, 보육

### 🟡 이벤트성 강등 규칙 (v7.2 신규)

다음 키워드가 공고 **제목**에 포함되면 🟢/🟠 판정이더라도 🟡로 강등(실제 자금 지원이 아닌 행사·컨설팅·오픈데이일 가능성 높음):

- **간담회, 네트워킹 데이, 컨설팅 데이, 오픈데이**

→ risk_flag=soft_downgrade_event 로 대시보드 표기.

## 🟠 AI 포지셔닝 (3순위)

- AX, LLM, AI대전환, AI바우처, AI 실증, 인공지능, 클라우드, 데이터, 딥테크

### 🟠 AX 강매칭 우회 규칙 (v7.2 신규)

다음 키워드가 공고 **제목**에 포함되면, 업종 한정 키워드가 일부 걸리더라도 일단 검토 대상으로 올린다:

- **AX, AX-버티컬, AX 버티컬, AI 버티컬, AI Vertical, AI 수직화, AI대전환, AI 대전환**

→ sinhon.life의 Claude API 기반 AI 비서가 'AI Vertical SaaS'로 포지셔닝 가능하므로, 단순 업종 한정 때문에 🔴로 떨어뜨리면 안 됨. 가점 플래그 부여 후 공고 원문 재확인.

## 🔴 즉시 제외

### 업종 한정 (true-red 목록 — v7.2에서 정비)

- **계속 red**: 반도체/팹리스/바이오/농업/에너지/방산/로봇/항공우주/관광/게임/가구/세라믹/해운물류/패션/의료기기/의료/제조/식품/농식품/수산/축산/원자력/조선/섬유/화학/철강/자동차부품/전통문화
- **v7.2 신규 red**: 생물자원/국립생물자원관/해양수산자원/유전자원 (명백한 업종 한정)
- **v7.2 red에서 제거**: 디바이스 / 그린 / 탄소중립 / ESG산업 — 이 키워드들은 `적용 영역` 또는 `가점 항목`으로 자주 등장하지만 업종 한정이 아님. false-red 방지를 위해 classify.py에서 일반 RED_INDUSTRY에서 빼고, **"탄소중립 전용" / "ESG 전용" / "그린산업 전용"** 같은 `전용` 한정사가 붙었을 때만 red로 판정

### 자격 한정

- OASIS/이민자/외국인/여성전용/여성창업/제대군인/자립청년/소상공인/귀농/귀촌/다문화

### 단계 불일치

- 후속지원/Series A/Series B/IPO/수출실적/업력 3년 초과(트랙 A 기준)/업력 5년/업력 7년/재도전/재창업

### 성격 불일치

- 수행기관/수행사/운영기관 모집/위탁운영/B2G/공공시장 진출/공공조달/조달시장/파트너사 모집

## 티어 판정 우선순위 (v7.2 보강)

충돌 시 다음 순서로 판정한다:

1. 🔴 즉시 제외 조건 중 자격·단계·성격 불일치 → red (다른 조건 무시)
2. **업종 한정 체크 → 단, 인천 앵커 기관 매칭 또는 AX 강매칭이 있으면 우회(industry_bypass risk_flag 부여)**
3. 지역·단계·업종이 모두 부합하면서 인천 앵커 기관 제목 매칭 → **green**
4. structured 공고 분야가 사업화/전국이면 → **green**
5. GREEN_KEYWORDS 매칭 → **green**
6. AI 관련이면 → **orange**
7. YELLOW_KEYWORDS 매칭 → **yellow**
8. 어느 것도 아니면 → **red**

**마지막 단계**: tier가 green/orange로 결정됐더라도 제목에 `간담회/네트워킹 데이/컨설팅 데이/오픈데이`가 있으면 → **yellow로 강등** (soft_downgrade_event)

## 용어 일치 (중요)

- "AX 수직화" = "AX 버티컬" = "AI Vertical SaaS" = "AI 대전환" (동일 개념, 혼용 허용)
- "Claude API 기반 AI 비서" = sinhon.life의 AI 포지셔닝 근거
- "예창패" = 예비창업패키지 (트랙 B 핵심 타겟)
- "초창패" = 초기창업패키지 (트랙 A 핵심 타겟)
- "인천 앵커" = 인천창조경제혁신센터/ICCE/인천TP/인천테크노파크/인천스타트업파크 (profile v7.2 신규)

## v7.2 교정 근거 (2026-04-19)

이번 업데이트는 2026-04-18 스냅샷 기준 false-red·false-green 감사에서 나온 3건의 실제 사례를 반영:

1. **false-red (구제)**: `2026년 ICCE x INNOPOLIS 창업스쿨 16기` (pbancSn=177179) — 인천 앵커 × 12대 전략기술 공고인데 이전 분류기에서 업종 한정으로 🔴 처리됨. 원인: INNOPOLIS 키워드 없이 본문의 "탄소중립기술 가점"을 업종 한정으로 오판. → 인천 앵커 우회 규칙으로 해결.
2. **false-red (구제)**: `초격차 스타트업 프로젝트 모두의 챌린지 AX - 버티컬` (pbancSn=176974) — 본문의 "디바이스·플랫폼 적용"을 업종 한정으로 오판. → "디바이스" RED에서 제거 + AX 강매칭 우회 규칙으로 해결.
3. **false-green (교정)**: `2026 생물자원 활용 창업벤처 지원을 위한 기업 간담회` (pbancSn=177116) — 이전 분류기에서 🟢로 잘못 올라옴. → "생물자원"을 RED_INDUSTRY에 추가 + "간담회" soft_downgrade 규칙으로 해결.
